# Java Modernization Lab

This repository contains a hands-on lab for modernizing legacy Java applications using IBM Bob.

## Getting Started

To perform the lab, please refer to the comprehensive lab guide:

📖 **[Lab Guide](lab-guide.md)**

## What You'll Learn

In this lab, you will:
- Experience IBM Bob features and capabilities
- Understand Bob built-in modes, custom modes, and MCP extensibility
- Explore Bob's comprehensive SDLC capabilities with a legacy Java application modernization use-case

## Lab Overview

Transform a **Legacy Struts 1.3** application to a modern **Spring Boot 3.x + React 18 + PostgreSQL** application:

- **Reverse Engineer** legacy code to generate comprehensive documentation and diagrams
- **Upgrade** Java version from 1.8 to 1.17
- **Modernize** from Struts to cloud-native Spring Boot
- **Generate** OpenShift/Kubernetes deployment artifacts

## Prerequisites

Before starting the lab, ensure you have:
1. IBM Bob IDE installed and configured
2. PlantUML Markdown Preview extension installed
3. Basic understanding of Java and web applications

## Quick Start

1. Open this project in IBM Bob IDE
2. Follow the step-by-step instructions in [lab-guide.md](lab-guide.md)
3. Complete the modernization journey from legacy to cloud-native
